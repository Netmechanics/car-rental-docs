= Τι νέο υπάρχει στη 1.1

Η γραμμή έκδοσης `1.1` συνοδεύεται από επέκταση και ωρίμανση της τεκμηρίωσης του `nmecar`, με κύρια έμφαση στην πρώτη ορατή έκδοση reporting μέσω του dashboard `Σύνοψη Επιχείρησης LITE`.

== Κύριες προσθήκες τεκμηρίωσης

Στην `1.1` προστίθενται ή οργανώνονται συστηματικότερα:

* νέα ενότητα xref:reporting/index.adoc[Reporting και Dashboards],
* dedicated περιγραφή του xref:reporting/lite-dashboard.adoc[Σύνοψη Επιχείρησης LITE],
* σαφής διαχωρισμός ανάμεσα στο τρέχον `LITE` scope και στη μελλοντική πλήρη suite dashboards / reports,
* πιο σαφής σύνδεση του dashboard integration με την επιχειρησιακή χρήση του,
* δομή release notes για τις επόμενες γραμμές έκδοσης.

== Στόχος της έκδοσης τεκμηρίωσης

Η `1.1` δίνει μεγαλύτερη έμφαση στους χρήστες που χρειάζονται:

* γρήγορη επιχειρησιακή εικόνα,
* καλύτερη κατανόηση των KPIs,
* καθαρή περιγραφή του ρόλου του `LITE` dashboard στη συνολική λειτουργία του συστήματος,
* ευκολότερη πλοήγηση σε feature-oriented documentation.

== Τρέχον reporting scope

Η `1.1` δεν δηλώνει ότι έχει ήδη ενεργοποιηθεί πλήρης σουίτα dashboards και reports.

Το reporting scope που παρουσιάζεται ως τρέχον παραγωγικό παραδοτέο είναι:

* ένα συμπαγές dashboard `LITE`,
* KPI cards,
* ένα ενιαίο trend chart,
* πίνακας `Top 5 Τοποθεσίες`,
* πίνακας `Action List`.

Τα πιο αναλυτικά dashboards και reports παραμένουν υλικό επόμενης έκδοσης.
