= Τι νέο υπάρχει στη 1.1
:revdate: Ιούνιος 2026

Η γραμμή έκδοσης `1.1` συνοδεύεται από ουσιαστικό εμπλουτισμό της τεκμηρίωσης του `nmecar`, ώστε να καλύπτει πιο ολοκληρωμένα τις βασικές λειτουργίες που χρησιμοποιούν καθημερινά οι χρήστες του συστήματος.

== Συνοπτικά

Στην `1.1` τεκμηριώνονται πιο καθαρά:

* το dashboard xref:reporting/lite-dashboard.adoc[Σύνοψη Επιχείρησης LITE],
* οι xref:admin/driver-age-charges.adoc[χρεώσεις ηλικίας οδηγού],
* η xref:booking/extras.adoc[διαθεσιμότητα extras ανά ομάδα οχημάτων],
* τα xref:technical/reports_email.adoc[reminder και thank-you emails],
* η xref:admin/payments.adoc[ροή επιβεβαίωσης τραπεζικής κατάθεσης],
* τα xref:admin/cms-seo-fields.adoc[SEO πεδία στα CMS posts/pages].

== Reporting και dashboard

Στην `1.1` προστίθεται νέα ενότητα xref:reporting/index.adoc[Reporting και Dashboards] με κεντρικό σημείο το dashboard xref:reporting/lite-dashboard.adoc[Σύνοψη Επιχείρησης LITE].

Ο χρήστης μπορεί να δει συγκεντρωμένα:

* βασικούς δείκτες καθημερινής λειτουργίας,
* συνοπτική εικόνα εσόδου και κρατήσεων,
* κορυφαίες τοποθεσίες,
* λίστα ενεργειών που χρειάζονται προσοχή.

== Τιμολόγηση και κράτηση

Η `1.1` αποτυπώνει πιο αναλυτικά βασικές λειτουργίες που επηρεάζουν το κόστος και τη συμπεριφορά της κράτησης:

* xref:admin/driver-age-charges.adoc[Χρεώσεις ηλικίας οδηγού]
  Ορίζονται ηλικιακές ζώνες, τύποι χρέωσης `FLAT` ή `PERCENTAGE`, κανόνες εγκυρότητας και ο τρόπος που η χρέωση αποτυπώνεται στην κράτηση.

* xref:booking/extras.adoc[Extras ανά ομάδα οχημάτων]
  Τα extras δεν αντιμετωπίζονται πλέον σαν καθολικά διαθέσιμα παντού, αλλά μπορούν να περιοριστούν σε συγκεκριμένες ομάδες οχημάτων.

* xref:booking/rentals.adoc[Ανάλυση κόστους ενοικίασης]
  Περιγράφονται καθαρότερα επιμέρους στοιχεία κόστους, όπως χρέωση ηλικίας οδηγού και ποσά που σχετίζονται με τρόπο πληρωμής.

== Emails και αυτοματισμοί

Η `1.1` περιγράφει πλέον πιο πλήρως τη ροή επικοινωνίας με τον πελάτη:

* xref:technical/reports_email.adoc[Πρότυπα email και reports]
  Καλύπτονται όχι μόνο τα contact / quote / rental confirmation emails αλλά και τα reminder και thank-you emails.

* xref:technical/scheduled-jobs.adoc[Προγραμματισμένες εργασίες]
  Περιγράφεται η εργασία follow-up emails που εκτελείται περιοδικά για τις αυτόματες αποστολές.

* xref:admin/settings-email.adoc[Ρυθμίσεις Email]
  Προστίθενται ξεχωριστές κατηγορίες ρυθμίσεων για reminder και thank-you emails.

* xref:admin/settings-system.adoc[Ρυθμίσεις Συστήματος]
  Καλύπτονται οι ενεργοποιήσεις και οι ημέρες αποστολής για τα follow-up emails.

== Πληρωμές και επιβεβαίωση κράτησης

Στην xref:admin/payments.adoc[ενότητα πληρωμών] περιγράφεται πιο καθαρά η ειδική ροή για `BANK_DEPOSIT`:

* η κράτηση παραμένει αρχικά σε κατάσταση `Κρατημένο`,
* ο χρήστης του backoffice επιβεβαιώνει χειροκίνητα την τραπεζική κατάθεση,
* η κράτηση μεταβαίνει σε `Μισθωμένο`,
* αποστέλλεται το κανονικό confirmation email,
* μπορεί να οριστεί ξεχωριστή περίοδος αναμονής για τέτοιες κρατήσεις.

== CMS και περιεχόμενο

Στην `1.1` προστίθεται η σελίδα xref:admin/cms-seo-fields.adoc[SEO πεδία CMS], όπου περιγράφονται:

* τα πεδία `meta title`, `meta description`, `meta keywords`,
* η πολυγλωσσική χρήση τους,
* η οργάνωσή τους στον editor των posts/pages,
* η αξιοποίησή τους από το public API και το front site.
