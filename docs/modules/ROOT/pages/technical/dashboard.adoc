= Πίνακας εργαλείων (Dashboard)

Δίνεται η δυνατότητα να συνδέσουμε την αρχική οθόνη της εφαρμογής με ένα dashboard του https://superset.apache.org/[Apache Superset,window=_blank].

image::lite-dashboard.png[alt="Σύνοψη Επιχείρησης LITE",width="100%"]

Το Superset είναι εξωτερική υπηρεσία, που υπάρχει στην υποδομή της Netmechanics και η εφαρμογή είναι παραμετροποιημένη να χρησιμοποιεί την υπηρεσία αυτή κατ' απαίτηση.

Για να ρυθμίσουμε τον πίνακα εργαλείων χρησιμοποιούμε τη σχετική xref:admin/settings-system.adoc#dashboard-embedded-id[ρύθμιση συστήματος].

Αν θέλουμε να απενεργοποιήσουμε τελείως τη δυνατότητα σύνδεσης της εφαρμογής με το Superset τότε ορίζουμε το application property `nmcr.dashboard.active` σε `false`.

[source,properties]
----
nmcr.dashboard.active = false
----

== Ρόλος της ενότητας dashboard στη 1.1

Στη γραμμή έκδοσης `1.1` το dashboard αντιμετωπίζεται ως μέρος μιας ευρύτερης ενότητας reporting και διοικητικής πληροφόρησης.

Η τεχνική σύνδεση εξακολουθεί να γίνεται μέσω Superset, όμως το περιεχόμενο που προβάλλεται οργανώνεται πλέον γύρω από:

* διοικητικούς δείκτες καθημερινής λειτουργίας,
* μια compact επιχειρησιακή εικόνα σε μορφή `LITE` dashboard,
* dashboard προβολή που βοηθά τον χρήστη να βλέπει γρήγορα τη συνολική εικόνα.

Για λειτουργική περιγραφή του reporting δείτε:

* xref:reporting/index.adoc[Reporting και Dashboards]
* xref:reporting/lite-dashboard.adoc[Σύνοψη Επιχείρησης LITE]
* xref:reporting/dashboards.adoc[Dashboards]
* xref:reporting/reports.adoc[Reports]

== Βασική ροή

[plantuml,dashboard-integration-flow,svg]
....
@startuml
title Ροή σύνδεσης εφαρμογής με dashboard

actor "Backoffice χρήστης" as User
rectangle "nmecar Backoffice" as Backoffice
database "System Settings" as Settings
cloud "Apache Superset" as Superset

User --> Backoffice : Ανοίγει αρχική οθόνη
Backoffice --> Settings : Διαβάζει dashboard embedded id
Backoffice --> Superset : Φορτώνει embedded dashboard
Superset --> Backoffice : Επιστρέφει dashboard view
Backoffice --> User : Προβολή dashboard

@enduml
....

== Προαπαιτούμενα

Πριν ενεργοποιηθεί η προβολή dashboard, πρέπει να έχουν επιβεβαιωθεί τα παρακάτω:

* η εξωτερική υπηρεσία Superset είναι διαθέσιμη,
* έχει δημιουργηθεί και δημοσιευθεί το αντίστοιχο dashboard στο Superset,
* έχει καταχωρηθεί το σωστό embedded id στις xref:admin/settings-system.adoc#dashboard-embedded-id[Ρυθμίσεις Συστήματος],
* το property `nmcr.dashboard.active` είναι ενεργό.
