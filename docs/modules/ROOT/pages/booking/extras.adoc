= Πρόσθετα

Τα πρόσθετα είναι επιπλέον υπηρεσίες ή εξοπλισμός που ο πελάτης μπορεί να επιλέξει να συμπεριλάβει στην ενοικίαση του.

Επιλέγονται από τον πελάτη κατά τη διάρκεια της διαδικασίας της ενοικίασης στο {app-front}.

Κάθε πρόσθετο έχει ένα σταθερό κόστος, το οποίο μπορεί να υπολογίζεται είτε ανά ημέρα είτε ανά κράτηση. Το κόστος αυτό μπορεί να οριστεί και ανά ομάδα οχημάτων για κάθε πρόσθετο.

Επιπλέον, κάθε πρόσθετο μπορεί να περιοριστεί ώστε να εμφανίζεται μόνο σε συγκεκριμένες ομάδες οχημάτων.

.Παράδειγμα υπολογισμού κόστους πρόσθετων
[%autowidth]
|===
s|Baby Seat| 1.00€ / ημέρα
s|Additional Driver | 15.00€ / κράτηση
s|Ημέρες ενοικίασης|10
s|Τιμή ημέρας|20.00€
2+|
s|Συνολικό κόστος ενοικίασης|45.00€ => 20.00 + (1.00 * 10) + 15.00
|===

== Πεδία Πρόσθετου

[options="header", cols="1m,3a"]
|===
|Πεδίο|Περιγραφή
|Τίτλος|Ο τίτλος του πρόσθετου είναι πολυγλωσσικό πεδίο.
|Εικόνα|Η εικόνα που μπορεί να έχει το πρόσθετο. Χρησιμοποιείται ανάλογα με το theme.
|Κόστος|Το σταθερό κόστος του πρόσθετου.
|Τύπος Κόστους|Ο τρόπος που θα υπολογίζεται το κόστος του πρόσθετου, *Ανά ημέρα* ή *Ανά κράτηση*

|Επιτρεπόμενες ομάδες οχημάτων|Προαιρετική allow-list ομάδων οχημάτων στις οποίες επιτρέπεται να εμφανιστεί το πρόσθετο.

|Ενεργό|Αν θα είναι ενεργό το πρόσθετο ή όχι. Αν δεν είναι ενεργό δε θα είναι διαθέσιμο στο {app-front}.
|===

== Διαθεσιμότητα ανά ομάδα οχημάτων

Η διαθεσιμότητα των extras και η τιμή τους είναι δύο διαφορετικά πράγματα:

* οι *Επιτρεπόμενες ομάδες οχημάτων* καθορίζουν αν ένα extra μπορεί να εμφανιστεί,
* το *Κόστος ομάδας οχημάτων* καθορίζει μόνο αν θα έχει διαφορετική τιμή.

Κανόνες:

* αν ένα extra έχει ρητά ορισμένες επιτρεπόμενες ομάδες, εμφανίζεται μόνο σε αυτές,
* αν ένα extra δεν έχει καμία επιτρεπόμενη ομάδα, λειτουργεί ως fallback / global extra για groups χωρίς ειδικά linked extras,
* το backend ελέγχει επίσης το reserve ώστε να απορρίψει invalid extras για λάθος vehicle group.

[plantuml,extra-availability-flow,svg]
....
@startuml
title Extras filtering ανά vehicle group

actor Customer
participant "Booking API" as API
participant "VehicleSelectService" as SelectService

Customer -> API : select vehicle / group
API -> SelectService : list extras for group

alt Υπάρχουν linked extras για το group
  SelectService -> SelectService : Επιστροφή μόνο linked extras
else Δεν υπάρχουν linked extras
  SelectService -> SelectService : Επιστροφή global/fallback extras
end

SelectService --> API : filtered extras
API --> Customer : visible extras list

@enduml
....
