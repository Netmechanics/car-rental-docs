= Διαδρομές

Ως διαδρομές ορίζονται ζεύγη τοποθεσιών (έναρξης και τερματισμού) που έχουν ένα συγκεκριμένο κόστος χρέωσης one-way.
Όταν ο πελάτης επιλέξει διαφορετικές τοποθεσίες παραλαβής και παράδοσης που αντιστοιχούν σε ορισμένη διαδρομή, το κόστος αυτό προστίθεται αυτόματα στη συνολική τιμή.

.Παράδειγμα
[example]
====
Διαδρομή: *Ηράκλειο Αεροδρόμιο → Ηράκλειο Κέντρο*, κόστος `15,00€`.

Αν ο πελάτης επιλέξει παραλαβή από το αεροδρόμιο και παράδοση στο κέντρο της πόλης,
το σύστημα θα προσθέσει 15,00€ στο συνολικό κόστος της ενοικίασης.
Αν επιλέξει αντίστροφη κατεύθυνση (Κέντρο → Αεροδρόμιο), αυτή θεωρείται ξεχωριστή διαδρομή
και πρέπει να οριστεί αντίστοιχα.
====

NOTE: Το τελικό ποσό χρέωσης εξαρτάται από την xref:admin/settings-system.adoc#locationsChargePolicy[πολιτική χρέωσης τοποθεσιών]. Ανάλογα με τη ρύθμιση, το κόστος μπορεί να ισχύει μόνο για ορισμένες παραλαβές/παραδόσεις (π.χ. μόνο για one-way).

== Πεδία Διαδρομής

[options="header", cols="1m,3a"]
|===
|Πεδίο|Περιγραφή
|Τοποθεσία έναρξης|Η τοποθεσία από την οποία ξεκινάει η διαδρομή.
|Τοποθεσία τερματισμού|Η τοποθεσία στην οποία τελειώνει η διαδρομή.
|Κόστος διαδρομής|Το κόστος της διαδρομής.
|Ενεργή|Αν η διαδρομή θα είναι ενεργή ή όχι.
|===
