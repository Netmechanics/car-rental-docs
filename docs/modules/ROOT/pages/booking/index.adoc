= Διαχείριση κρατήσεων

Η ενότητα *Κρατήσεις* περιλαμβάνει όλες τις παραμέτρους που διαμορφώνουν τη διαδικασία ενοικίασης —
από τον ορισμό τοποθεσιών και διαδρομών, μέχρι τους κανόνες τιμολόγησης, εκπτώσεων και διαθεσιμότητας.

Η σωστή παραμετροποίηση αυτής της ενότητας είναι προϋπόθεση για την ορθή λειτουργία της αναζήτησης
και των κρατήσεων στο front-end της εφαρμογής.

Η ενότητα αυτή περιλαμβάνει:

* xref:booking/locations.adoc[Τοποθεσίες] — σημεία παραλαβής και παράδοσης, κόστη, ομαδοποίηση.
* xref:booking/routes.adoc[Διαδρομές] — ζεύγη τοποθεσιών με χρέωση one-way διαδρομής.
* xref:booking/rental-search-logic.adoc[Λογική αναζήτησης] — κανόνες διάρκειας, extra hours, αποτελέσματα αναζήτησης.
* xref:booking/seasons.adoc[Σεζόν] — ορισμός χρονικών περιόδων τιμολόγησης.
* xref:booking/coupons.adoc[Κουπόνια] — κωδικοί έκπτωσης, διάρκεια ισχύος, regex validation.
* xref:booking/insurances.adoc[Ασφάλειες] — μοντέλα χρέωσης per-day και per-period.
* xref:booking/discounts.adoc[Εκπτώσεις] — τύποι Α/Β, κανόνες προτεραιότητας, παραδείγματα.
* xref:booking/extras.adoc[Πρόσθετα] — επιπλέον υπηρεσίες ανά κράτηση ή ημέρα.
* xref:booking/rentals.adoc[Ενοικιάσεις] — προβολή και διαχείριση υπαρχουσών κρατήσεων.
