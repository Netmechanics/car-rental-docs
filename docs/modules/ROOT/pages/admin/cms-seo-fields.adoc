= SEO πεδία CMS

Τα SEO πεδία στα CMS posts/pages επιτρέπουν στο backoffice να ορίζει ξεχωριστά metadata για την παρουσίαση του περιεχομένου στο public site και στις μηχανές αναζήτησης.

Η λειτουργία αυτή χρησιμοποιείται κυρίως για:

* βελτιωμένο τίτλο σε search results,
* πιο ακριβή περιγραφή σε social/share previews και search snippets,
* καλύτερη οργάνωση περιεχομένου ανά γλώσσα.

== Διαθέσιμα πεδία

[options="header", cols="1m,3a"]
|===
|Πεδίο|Περιγραφή
|Meta title|Ο τίτλος SEO του περιεχομένου.
|Meta description|Σύντομη περιγραφή για search snippets και previews.
|Meta keywords|Λέξεις-κλειδιά για οργανωμένο SEO metadata.
|===

Και τα τρία πεδία είναι πολυγλωσσικά.

== Οργάνωση στο backoffice

image:cms_editor.png[]

Στον editor των posts/pages:

* τα γενικά στοιχεία παραμένουν στο βασικό form,
* το `Text`, το `Small text` και το `SEO` εμφανίζονται σε ξεχωριστά tabs,
* τα media παραμένουν σε ξεχωριστό panel.

Αυτό κάνει την επεξεργασία περιεχομένου πιο καθαρή, ειδικά όταν το post έχει πολλές γλώσσες και συνδυάζει editorial και SEO πεδία.

== Χρήση στο public API

Τα SEO fields επιστρέφονται από το Posts API σαν localized values, ώστε το front site να μπορεί να αποδίδει σωστά:

* `metaTitle`
* `metaDescription`
* `metaKeywords`

Η πληροφορία αυτή είναι διαθέσιμη και στα list και στα detail responses των posts.

== Καλές πρακτικές

* Χρησιμοποιείτε τίτλο SEO όταν θέλετε κάτι πιο σύντομο ή πιο εμπορικό από τον κύριο τίτλο της σελίδας.
* Συμπληρώνετε περιγραφή που να στέκεται και μόνη της εκτός του site, π.χ. σε αποτέλεσμα αναζήτησης.
* Όταν το περιεχόμενο είναι πολύγλωσσο, φροντίζετε να υπάρχουν τιμές σε κάθε ενεργή γλώσσα.
