= Διαχείριση συστήματος

Η ενότητα *Διαχείριση* περιλαμβάνει τις βασικές παραμέτρους διαμόρφωσης του συστήματος.
Εδώ ορίζονται τα δεδομένα αναφοράς (μοντέλα, τύποι, χαρακτηριστικά οχημάτων),
οι κανόνες πρόσβασης και πληρωμών, οι αλγόριθμοι τιμολόγησης,
καθώς και οι ρυθμίσεις ολοκλήρωσης με εξωτερικές υπηρεσίες (email, Google, CMS).

CAUTION: Οι αλλαγές σε αυτή την ενότητα επηρεάζουν τη λειτουργία ολόκληρης της εφαρμογής.
Κάθε παραμετροποίηση σε παραγωγικό περιβάλλον πρέπει να γίνεται με προσοχή.

Η ενότητα αυτή περιλαμβάνει:

* xref:admin/vehicle_models.adoc[Μοντέλα], xref:admin/vehicle_types.adoc[Τύποι] και xref:admin/vehicle_features.adoc[Χαρακτηριστικά οχημάτων] — δεδομένα αναφοράς για τον στόλο.
* xref:admin/tags.adoc[Ετικέτες] — labels για ομάδες και οχήματα, ορατά στο front-end.
* xref:admin/roles.adoc[Ρόλοι] — δικαιώματα χρηστών backoffice.
* xref:admin/payments.adoc[Τρόποι πληρωμής] — ορισμός μεθόδων, παρόχων και τελών.
* xref:admin/driver-age-charges.adoc[Χρεώσεις ηλικίας οδηγού] — ζώνες ηλικίας και υπολογισμός χρέωσης.
* xref:admin/price-rounding.adoc[Στρογγυλοποίηση τιμών] — κλίμακα και δεκαδικά ψηφία κόστους.
* xref:admin/settings-system.adoc[Ρυθμίσεις συστήματος] — locales, ώρες, Google API, scheduled jobs και άλλα.
* xref:admin/settings-email.adoc[Ρυθμίσεις email] — κατηγορίες αποστολής, αποδέκτες, πρότυπα.
* xref:admin/cms-seo-fields.adoc[CMS / SEO] — πολύγλωσσα μεταδεδομένα για το δημόσιο API.
