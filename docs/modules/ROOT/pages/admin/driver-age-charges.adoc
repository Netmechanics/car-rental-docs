= Χρεώσεις ηλικίας οδηγού

Οι χρεώσεις ηλικίας οδηγού επιτρέπουν στο σύστημα να εφαρμόζει επιπλέον κόστος όταν η ηλικία του πελάτη, κατά την ημερομηνία παραλαβής, εμπίπτει σε συγκεκριμένη ηλικιακή ζώνη.

Η λειτουργία είναι παραμετρική και διαχειρίζεται από το backoffice.

== Τι λύνει

Η δυνατότητα αυτή είναι χρήσιμη όταν η εμπορική πολιτική διαφέρει για:

* νέους οδηγούς,
* μεγαλύτερες ηλικίες οδηγών,
* ειδικές κατηγορίες πελατών με αυξημένο ρίσκο ή ιδιαίτερη τιμολογιακή πολιτική.

== Πεδία ηλικιακής ζώνης

[options="header", cols="1m,3a"]
|===
|Πεδίο|Περιγραφή
|Τίτλος|Πολυγλωσσικός τίτλος της χρέωσης, π.χ. `Young driver surcharge`.
|Ηλικία από|Κατώτατο όριο ηλικίας. Το όριο συμπεριλαμβάνεται.
|Ηλικία έως|Ανώτατο όριο ηλικίας. Το όριο συμπεριλαμβάνεται.
|Τύπος χρέωσης|`FLAT` ή `PERCENTAGE`.
|Ποσό|Σταθερό ποσό ή ποσοστό, ανάλογα με τον τύπο.
|Ενεργό|Αν η ζώνη συμμετέχει στον υπολογισμό ή όχι.
|===

== Κανόνες εγκυρότητας

Ισχύουν οι παρακάτω κανόνες:

* το `Ηλικία από` δεν μπορεί να είναι μεγαλύτερο από το `Ηλικία έως`,
* οι ενεργές ηλικιακές ζώνες δεν επιτρέπεται να επικαλύπτονται,
* οι ανενεργές ζώνες δεν επηρεάζουν τις υπόλοιπες,
* αν μια ανενεργή ζώνη ενεργοποιηθεί, τότε ελέγχεται ξανά για overlap.

== Τρόποι υπολογισμού

=== FLAT

Η χρέωση είναι σταθερό ποσό.

Παράδειγμα:

* ζώνη `18-20`
* τύπος `FLAT`
* ποσό `35.00`

Σε αυτή την περίπτωση προστίθενται `35.00€` στην κράτηση.

=== PERCENTAGE

Η χρέωση υπολογίζεται ποσοστιαία πάνω στο rounded vehicle season price, πριν προστεθούν extra hours.

Δεν υπολογίζεται πάνω σε:

* extras,
* insurance,
* pickup / drop-off costs,
* route cost,
* payment method cost,
* VAT,
* coupon discount.

[NOTE]
====
Το frontend δεν πρέπει να επανυπολογίζει το percentage. Το ποσό που επιστρέφει το backend είναι το authoritative ποσό της χρέωσης.
====

== Συμπεριφορά στη διαδικασία κράτησης

Η ηλικία υπολογίζεται από:

* την ημερομηνία γέννησης του πελάτη,
* την ημερομηνία παραλαβής.

Αν βρεθεί ενεργή ηλικιακή ζώνη που καλύπτει την ηλικία του πελάτη:

* η χρέωση επιστρέφεται από το booking API,
* προστίθεται στο cost breakdown,
* ελέγχεται ξανά από τον server κατά το reserve,
* αποθηκεύεται snapshot στην ενοικίαση.

== Snapshot στην ενοικίαση

Στην ενοικίαση αποθηκεύονται τα εξής:

* τίτλος χρέωσης ηλικίας οδηγού,
* ποσό χρέωσης ηλικίας οδηγού.

Ο λόγος είναι να διατηρείται η ιστορική αλήθεια της κράτησης, ακόμα κι αν στο μέλλον αλλάξει η ηλικιακή ζώνη ή η μετάφρασή της.
