= Οχήματα

Τα οχήματα περιλαμβάνουν αυτοκίνητα, φορτηγά, μοτοποδήλατα και κάθε άλλο μεταφορικό μέσο που παρέχει η εταιρεία προς ενοικίαση.

Κάθε όχημα ανήκει σε μια *ομάδα οχημάτων* και μπορεί να έχει περισσότερες από μία *πινακίδες κυκλοφορίας*.
Ο πελάτης αναζητά και επιλέγει βάσει ομάδας — όταν ολοκληρωθεί η κράτηση, το σύστημα δεσμεύει αυτόματα μια συγκεκριμένη πινακίδα.
Για τον λόγο αυτό, η διαθεσιμότητα και η κατάσταση παρακολουθούνται σε επίπεδο πινακίδας, και όχι σε επίπεδο οχήματος.

== Πεδία Πινακίδων
[options="header",cols="1m,3a"]
|===
|Πεδίο|Περιγραφή
|Αριθμός πινακίδας|Ο αριθμός πινακίδας του οχήματος.
|Αριθμός αδείας|Ο αριθμός άδειας κυκλοφορίας του οχήματος.
|Κατάσταση|Η κατάσταση του οχήματος.

Δέχεται τιμές:

* *Άγνωστη (Unknown):* Αρχική κατάσταση — δεν έχει οριστεί ακόμα.
* *Διαθέσιμο (Available):* Το όχημα είναι ελεύθερο για κράτηση.
* *Δεσμευμένο (Reserved):* Έχει δεσμευτεί κατά τη διαδικασία κράτησης, πριν την ολοκλήρωση της πληρωμής.
* *Μισθωμένο (Rented):* Σε ενεργή ενοικίαση μετά από επιτυχή πληρωμή.
* *Στο μηχανικό (In Service):* Εκτός κυκλοφορίας για συντήρηση ή επισκευή.

|===

== Κύκλος ζωής πινακίδας

Ορισμένες μεταβάσεις γίνονται αυτόματα από το σύστημα κατά τη ροή ενοικίασης.
Οι υπόλοιπες γίνονται χειροκίνητα από τον διαχειριστή.

[plantuml,plate-lifecycle,svg]
....
@startuml
hide empty description
skinparam defaultFontName Arial
skinparam state {
  BackgroundColor #EEF4FF
  BorderColor #4A6FA5
  ArrowColor #4A6FA5
}

state "Άγνωστη" as Unknown
state "Διαθέσιμο" as Available
state "Δεσμευμένο" as Reserved
state "Μισθωμένο" as Rented
state "Στο μηχανικό" as InService

[*] --> Unknown
Unknown --> Available : χειροκίνητη αλλαγή
Available --> Reserved : έναρξη κράτησης\n(πριν πληρωμή)
Reserved --> Available : ακύρωση / εγκατάλειψη\n(AbandonedRentalsJob)
Reserved --> Rented : επιτυχής πληρωμή
Rented --> Available : ολοκλήρωση ενοικίασης\n(FinishRentalsJob)
Available --> InService : χειροκίνητη αλλαγή
InService --> Available : χειροκίνητη αλλαγή
@enduml
....

NOTE: Οι μεταβάσεις *Δεσμευμένο → Διαθέσιμο* και *Μισθωμένο → Διαθέσιμο* γίνονται αποκλειστικά από προγραμματισμένες εργασίες. Δείτε xref:technical/scheduled-jobs.adoc[].
