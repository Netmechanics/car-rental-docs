= Διαχείριση στόλου

Η ενότητα *Στόλος* αφορά την οργάνωση και παρακολούθηση των οχημάτων της επιχείρησης.

Τα οχήματα οργανώνονται σε *ομάδες οχημάτων* (π.χ. «Μικρά», «SUV», «Minivan»).
Κάθε ομάδα ορίζει τη λογική τιμολόγησης ανά σεζόν, τα διαθέσιμα πρόσθετα, τις ασφάλειες και τις εκπτώσεις που ισχύουν για τα οχήματά της.
Ο πελάτης αναζητά και επιλέγει βάσει ομάδας — η τελική ανάθεση γίνεται σε επίπεδο *πινακίδας κυκλοφορίας*.

Η ενότητα αυτή περιλαμβάνει:

* xref:fleet/vehicle_groups.adoc[Ομάδες οχημάτων] — δημιουργία ομάδων, τιμολόγηση ανά σεζόν, περίοδοι μη διαθεσιμότητας, tags, εικόνες.
* xref:fleet/vehicles.adoc[Οχήματα] — καταχώριση οχημάτων ανά μοντέλο, διαχείριση πινακίδων, κατάσταση, τοποθεσία και εικόνες.
