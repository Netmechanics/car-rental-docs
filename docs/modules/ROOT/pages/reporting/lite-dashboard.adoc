= Σύνοψη Επιχείρησης LITE

Το dashboard `Σύνοψη Επιχείρησης LITE` είναι το πρώτο οργανωμένο reporting deliverable που παρουσιάζεται στη γραμμή `1.1`.

Στόχος του είναι να δίνει άμεση πρώτη αξία στον τελικό χρήστη, χωρίς να φορτώνει την οθόνη με υπερβολική πληροφορία.

Πρακτικά λειτουργεί ως:

* κοινό entry point για sales, operations και διοίκηση,
* single-screen dashboard με μικρό αλλά ουσιώδες επιχειρησιακό scope.

image::lite-dashboard.png[alt="Σύνοψη Επιχείρησης LITE",width="100%"]

== Τι περιλαμβάνει

Η πρώτη έκδοση του `LITE` dashboard περιλαμβάνει μόνο:

* KPI cards,
* ένα ενιαίο trend chart,
* πίνακα `Top 5 Τοποθεσίες`,
* πίνακα `Action List`.

Δεν περιλαμβάνει σε αυτό το build:

* drill-through navigation,
* extra tabs,
* δεύτερη σελίδα dashboard,
* πλήρες location breakdown,
* πλήρες operations board,
* advanced forecast ή pricing layers.

== Επιχειρησιακός στόχος

Ο χρήστης πρέπει να μπορεί να καταλάβει σε λιγότερο από ένα λεπτό:

* τι κίνηση υπάρχει,
* τι αξία παράγει,
* πόση πίεση δέχεται ο στόλος,
* ποια σημεία θέλουν άμεση προσοχή.

== Προτεινόμενα charts

Το `LITE` dashboard κλειδώνει scope σε `11` charts:

1. `Κρατήσεις Σήμερα`
2. `Παραλαβές Σήμερα`
3. `Ανοιχτά Rentals`
4. `Έτοιμα για Check-in`
5. `Αναμενόμενο Έσοδο`
6. `Υπόλοιπο προς Είσπραξη`
7. `Αξιοποίηση Στόλου`
8. `Operational Alerts`
9. `Κρατήσεις / Έσοδο / Αξιοποίηση`
10. `Top 5 Τοποθεσίες`

== Filters και interactions

Η πρώτη έκδοση πρέπει να μείνει απλή.

Διαθέσιμα φίλτρα:

* `Ημερομηνία αναφοράς`
* `Horizon ημερών`

Στο τρέχον UI του Superset τα φίλτρα εμφανίζονται στο αριστερό filter panel και όχι ως ξεχωριστή πάνω ζώνη μέσα στο σώμα του dashboard.

Interaction policy:

* τα KPI cards δεν χρειάζεται να έχουν drill-down στο v1,
* οι πίνακες παραμένουν read-only summary lists,
* το dashboard χρησιμοποιείται για γρήγορη εποπτεία και όχι για βαθιά ανάλυση μέσα από την ίδια οθόνη.

== Layout

Το dashboard πρέπει να διαβάζεται σε μία οθόνη laptop χωρίς έντονο scrolling.

Προτεινόμενη διάταξη:

* πάνω ζώνη με τίτλο και σύντομο explanatory text
* αριστερό filter panel του Superset
* KPI strip με `8` Big Number charts σε `2` σειρές των `4`
* ένα μεγάλο mixed trend chart full width
* δύο summary tables στο κάτω μέρος
